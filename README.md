# Secure Password Generator

## Project Description
This project is a Secure Password Generator created for the Information Security final project.  
The main purpose of this program is to generate strong and secure passwords for users.  
It helps users create passwords that are difficult to guess and can improve account security.

## Objectives
- Generate strong random passwords
- Allow users to choose password length
- Include uppercase letters
- Include lowercase letters
- Include numbers
- Include special characters
- Help users improve password security

## Why This Project is Important
Passwords are one of the most common ways to protect personal accounts and systems.  
Weak passwords can be guessed easily and may cause security problems.  
This project helps solve that issue by creating stronger passwords.

## Features
- User can choose the length of the password
- User can generate a random secure password
- Password can include:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Symbols
- Simple and easy to use
